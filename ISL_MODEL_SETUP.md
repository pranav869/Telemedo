# Instructions to Download and Place the ISL Model

1. Go to: https://github.com/harshpatel2/Indian-Sign-Language-Recognition-using-CNN
2. Download the pre-trained model file (usually named `Trained_model.h5` or similar) from the repository's releases or model folder.
3. Place the downloaded `.h5` file in your project root directory and rename it to `isl_model.h5` for consistency.
4. (Optional) Download the class labels file (e.g., `labels.txt` or use the mapping from the repo's code) and place it in the root as `isl_labels.txt`.

Once the model is in place, the backend will be able to load and use it for ISL predictions.
